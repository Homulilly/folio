# 更新日志

本项目所有重要变更记录于此。格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/),版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.2.3] - 2026-06-22

### 修复

- **macOS 应用图标改为圆角 squircle**:此前是满铺白底方图,在 Dock / 访达显示为未裁剪的大白方块(macOS 不会自动为图标加圆角)。现把图标重塑为标准 macOS 模板——圆角磁贴 + 四周透明留白 + 轻微投影,重新生成 `icon.icns`。Windows `.ico`(满铺方图)不受影响。

## [0.2.2] - 2026-06-22

### 新增

- **设置页新增「关于」段落**:展示应用图标、版本号与构建信息(Electron / Chromium / 平台架构);内含「致谢」按钮,弹窗列出 Folio 依赖的开源项目,链接经系统浏览器打开。

### 变更

- **应用包标识(appId)改为 `me.nep.folio`**:替换占位的反向域名,作为 macOS bundle ID 与 Windows AUMID,下次打包生效。

## [0.2.1] - 2026-06-21

### 新增

- **EXIF 概览新增字段**:描述 / 作者 / 版权 / 注释 / 唯一 ID;其中 **描述 / 作者 / 版权 在主值(IFD0)为空时回退读取缩略图 IFD1**(标签 0x010E / 0x013B / 0x8298)。
- **EXIF「原始」视图改为原始 JSON**:展示完整的 `{ 分组: { 字段: 值 } }` JSON(搜索仍可过滤)。
- **EXIF「分组」视图字段名中文化**:常见拍摄 / 容器 / ICC 等标签按界面语言显示中文;原始标签名仍可在 hover 与原始 JSON 中查看。
- **侧栏默认收起 + 启动状态可设置**:设置页「浏览」新增「启动时收起侧栏」开关。

### 变更

- **EXIF 概览不再显示 GPS 坐标**:改为橙色「包含 GPS 信息」徽标(与时间同色),坐标在 hover 时显示。
- **格式转换前预检**:当前系统不支持作为转换输入的格式(ICO,以及 Windows 上的 HEIC / JPEG XL)在点击转换时即提示,而非选完参数后才失败。
- **转换质量改为滑块 + 数字框**;AVIF 的速度档收进与质量同一行。
- **EXIF 抽屉底部按钮并排**:擦除 EXIF / 复制 JSON 两个等宽按钮,擦除图标改为垃圾桶。
- **多图滚轮缩放更顺滑**:对每帧滚轮增量削峰,高速(自由滚)滚轮不再造成剧烈跳变。

### 修复

- **多图模式每格独立缩放**:此前缩放是单一全局值,切换焦点会把上一格的缩放带到新格、上一格回退 fit;现各格记住自己的缩放(同步缩放仍共享)。
- **安全硬化**:外部链接仅放行 http/https(封堵 `file://` / `smb://` / 自定义协议),并锁定窗口导航不被改写到其它来源。

## [0.2.0] - 2026-06-21

### 新增

- **macOS 浏览 JPEG XL**:经系统 `sips`(ImageIO)解码显示与缩放(沿用 HEIC 的兜底管线)。
- **格式转换支持 HEIC / HEIF / JPEG XL / BMP 作为输入**(macOS):sharp 无法解码时,经统一解码层 `services/decode.ts` 用系统解码器转中间 PNG 再编码到目标格式,修复了「能看不能转」。
- **文件列表纳入 SVG / ICO**:由 Chromium 原生显示;sharp 不可栅格化时缩略图回退到原图。
- **工具栏「关闭文件夹(返回首页)」**:清空当前队列回到起始页。

### 变更

- 格式转换图标改为魔法棒(wand-sparkles)。
- 下载产物名加 `v` 前缀(如 `Folio-v0.2.0-arm64.dmg`),与发布 tag 对齐。

### 修复

- 「暂不支持」占位 / 加载失败提示在单图视图中错位,改回居中。

### 已知限制

- ICO 仅显示,不作为格式转换输入。
- Windows / Linux 的 HEIC / HEIF / JPEG XL 暂不支持(显示与转换均依赖系统解码器 / 后续随包 djxl·libheif)。

## [0.1.1] - 2026-06-21

### 修复

- 单图放大后,溢出到画布左侧 / 顶部的内容无法拖动或滚动查看(改用 `margin:auto` 居中,避免 flex 居中裁剪前缘溢出,使两侧溢出都可滚动)。

## [0.1.0] - 2026-06-21

首个 MVP 版本(里程碑 M0–M7):轻量、快速、注重隐私的跨平台看图工具,内置 Exif 查看/擦除、批量重命名、格式转换与**多图并列浏览**。

### 新增

**看图与浏览**
- 单图查看:适应窗口 / 原始尺寸 / 缩放 / 旋转 + 拖拽平移,鼠标滚轮以光标为锚点缩放。
- 打开方式:文件 / 文件夹对话框、拖拽文件或文件夹入窗、聚焦单文件时扫描其所在目录。
- 导航:上一张 / 下一张 / 首张 / 末张 / 随机;到末图按「下一张」可提示载入下一个含图同级文件夹。
- 文件夹浏览:队列侧栏可浏览同级 / 父级目录、刷新当前目录、载入下一个文件夹。
- 排序:名称 / 修改时间 / 创建时间 / 大小 / 格式。
- 文件操作:删除到回收站(可配置确认)、在文件管理器中显示、复制路径、复制图片、最近文件夹、原生文件拖出。

**多图并列浏览(差异化核心)**
- 单 / 双 / 三 / 四宫格模式,多种布局变体(双图横竖、三图左大右二 / 三列、四宫格)。
- 分组切换、焦点格高亮、点击 / 数字键 / Tab 切换焦点、`Enter` 临时放大为单图。
- 同步缩放、循环浏览;离屏格子卸载以释放解码内存,大图栅格自动用轻量预览。

**Exif 元信息**
- 查看:摘要 / 分组 / 原始三视图,按字段名或值搜索、按分组筛选,复制单字段或导出完整 JSON。
- 擦除 / 隐私清理:按分类(GPS / 设备 / 唯一 ID / 时间 / 软件 / 缩略图 / 描述 / 作者版权)与自定义标签擦除;内置预设(隐私 / 分享 / 完全清理 / 保留版权);擦除前差异预览、擦除后重读验证。
- 自动模式:首张擦除后可应用到同目录其他图片(会话级),始终导出新文件。

**批处理与文件操作**
- 任务调度器:进度 / 暂停 / 继续 / 取消 / 失败重试 / 导出日志,批处理任务页,任务历史跨重启恢复。
- 保存到目标文件夹:命名模板(`{name}{ext}{md5}{sha1}{date}{time}{width}{height}{nr:001}` 等)、冲突策略(skip / overwrite / 自动编号 / md5 比对)。
- 快速保存:可配置多个目标文件夹,单目标按 `T` 直接保存,多目标弹出选择菜单。
- 批量重命名:替换 / 删除 / 序号三种模式,实时 dry-run 预览,检测非法字符 / 重名 / 循环冲突(A→B,B→A),两阶段执行失败回滚。
- 格式转换:JPEG / PNG / WebP / AVIF / TIFF,可调质量 / 压缩参数,保留 Exif 与 ICC。

**性能与缓存**
- 缩略图(256px)/ 预览(2048px)变体由 sharp 生成 webp,落盘 + SQLite 索引,按路径 + 尺寸 + mtime 失效,LRU 按缓存大小上限淘汰。
- Exif 摘要与文件哈希持久化缓存(L2),跨重启免重算。
- 队列侧栏虚拟化:仅渲染可视区行,大文件夹不卡顿。
- 真实尺寸经 sharp 读文件头获取,大图按尺寸择源以控内存。

**设置与隐私**
- 设置写入 `settings.json`(原子写),启动先读后渲染(语言无闪烁);界面语言(简体中文 / 英文)即时切换。
- 记住上次:排序、多图模式、循环 / 同步缩放、快速保存规则、默认擦除规则。
- 诊断:崩溃报告器(转储留本机)+ 主进程错误日志,一键打开日志文件夹;**全部离线,绝不上传**。
- 安全基线:默认导出新文件、绝不覆盖原图;就地覆盖二次确认;批处理有预览 / 确认 / 日志。

**界面与窗口**
- macOS / Windows 自定义标题栏;沉浸式全屏(边缘滑出控件)。
- 队列侧栏缩略图(含 BMP / SVG / ICO 等浏览器可解码格式回退原图显示)。

**格式支持**
- 浏览器原生解码:JPEG / PNG / GIF / WebP / BMP / AVIF / SVG / ICO。
- 经 sharp 预览:TIFF 等。
- HEIC / HEIF:**macOS** 经系统 `sips` 兜底显示与缩放(真实格式按 magic bytes 识别,扩展名说谎也能正确处理)。
- 真实格式按文件头(magic bytes)嗅探,扩展名仅用于枚举。

**工程 / 打包**
- pnpm workspace monorepo + electron-vite 三进程;核心纯逻辑在 `packages/core`(Vitest 覆盖)。
- 自定义协议 `gv-img://` 流式传图,绝不经 base64 IPC。
- electron-builder 打包:macOS **未签名** DMG 已验证可启动(ad-hoc 签名兜底);应用图标(mac `.icns` / win `.ico`)。

### 已知限制

- **JPEG XL**:预置 libvips 无解码器,暂不支持,显示「暂不支持」提示。
- **Windows HEIC**:暂不支持(依赖系统编解码器,后续接入)。
- **代码签名 / 公证**:暂为未签名构建,首次打开需绕过 Gatekeeper / SmartScreen 提示;待接入 Developer ID / Authenticode。
- **Windows / Linux 产物**:配置就位但未在对应平台实跑(pnpm 不自动安装其它平台的 sharp 预编译二进制)。
- 详见 `docs/mvp-tasks.md` 与 `docs/packaging.md`。
