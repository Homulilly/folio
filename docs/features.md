# Folio 已实现功能

> 本文记录 **当前代码已落地** 的功能(M1 基础看图 · M2 多图并列 · M3 Exif 查看 · M4 Exif 擦除/批处理/自动模式)。
> 完整需求/路线图见 [`prd.md`](./prd.md),里程碑进度见 [`mvp-tasks.md`](./mvp-tasks.md)。

## 打开与队列

- **打开方式**:打开文件、打开文件夹、拖拽文件/文件夹到窗口。
  - 打开单张图片 → 扫描其所在文件夹并定位到该图。
  - 打开文件夹 → 非递归扫描该目录下受支持的图片。
  - 拖拽多个路径 → 只收其中受支持的图片。
- **队列侧栏**:左侧图片列表,显示格式徽标、文件名、大小;点击切换;当前项高亮,多图模式下当前组整组高亮。
  - 侧栏可 **收起/展开**(工具栏最左侧的侧栏按钮),收起后画布占满宽度。
- **最近文件夹**:记录并可清空(主进程持久化)。
- **排序**:名称、修改时间、创建时间、大小、格式(升/降序);切换排序后保持当前焦点图。

## 单图查看

- **适应窗口(Fit)**:按 **宽高同时计算** 缩放,确保整图完整可见,且不超过原始尺寸(100%);90°/270° 旋转时按旋转后的边界计算。
- **缩放**:
  - 工具栏按钮、`+/-`、`0`(原始尺寸)、`空格`(Fit/原始切换)。
  - **鼠标滚轮缩放**:滚轮即缩放,按当前显示比例平滑增减;从 Fit 进入缩放时 **从实际适应比例开始**(超大图不会突跳到 100%)。
- **平移**:放大后按住鼠标拖动平移,横向纵向均可。
- **旋转**:右转(整圆箭头图标);旁边的 **重置方向** 按钮在旋转后才激活。
- **切换图片箭头**:鼠标移动到画布 **左右边缘附近** 时浮现半透明箭头,点击上一张/下一张;到首/尾时对应箭头隐藏(开启循环则不隐藏)。
- **自定义滚动条**:浅灰色、半透明、仅滑块,悬浮在图片之上;隐藏原生滚动条,滚动条出现/消失不引起布局变动;可拖动。队列侧栏使用同款滚动条。

## 多图并列(差异化核心)

- **模式**:单图 / 双图 / 三图 / 四宫格;工具栏按钮、`V` 循环、`Ctrl/Cmd+1..4` 切换。
- **布局变体**:双图(横/竖)、三图(左大右二 / 三列等宽)、四宫格 2×2;有变体的模式可用工具栏「Swap layout」切换。
- **分组浏览**:按模式的图片数成组步进;末尾不足时空白格显示「No more images」;可开启 **循环浏览**。
  - **网格内滚轮**:在多列网格上滚动滚轮 **成组切换**(下一组/上一组,带节流)。
- **焦点图**:点击格子、数字键 `1-4`、`Tab`/`Shift+Tab` 选择焦点,焦点格蓝色边框;焦点图驱动状态栏与文件操作。
- **焦点单图**:`Enter` 或 **双击格子** 将焦点图临时放大为单图;`Esc` 或 **右下角半透明返回按钮** 回到网格。进入/返回都重置为 Fit。
- **缩放联动**:`S` 或工具栏开关切换 **同步缩放**——开启时所有格同步缩放,关闭时仅焦点格缩放。
- **旋转**:多列下右转 **对所有格生效**,并 **带入焦点单图**(共享旋转状态)。
- **状态栏**:显示当前模式、组范围(第 X–Y 张 / 共 N)、焦点图文件名/尺寸/格式/缩放、Sync 指示。
- **切换重置**:切换视图数量(单/双/三/四)、进出焦点单图时,视图重置为 Fit。

## 格式与元信息

- **真实格式识别**:按 **magic bytes(文件头)** 判断真实格式,而非扩展名——扩展名说谎的图片(如 `.png` 实为 JPEG)也能正确判定与显示。
  - 覆盖 JPEG/PNG/GIF/WebP/BMP/TIFF/AVIF/HEIC/HEIF/JPEG XL;纯检测逻辑在 `@folio/image-processing`(含单测)。
  - `gv-img://` 协议据此返回正确的 `Content-Type`;渲染层据此判断可渲染性与格式标签。
- **渲染范围**:Chromium 原生可解码的格式直接显示;暂不支持的格式(TIFF/HEIC/HEIF/JXL 等)显示占位提示,待 M6/M7 接入 sharp 预览。

## Exif 查看(M3)

- **Exif 抽屉**:右侧抽屉(工具栏信息按钮或 `I` 开关);多图模式下 **跟随焦点图**。
- **三视图**:概览(Summary,相机/镜头/曝光/GPS 等要点)/ 分组(按 EXIF·GPS·XMP·IPTC·File 等分组)/ 原始(Raw 全字段)。
- **搜索**:按字段名或值过滤;输入分组名展开整组。
- **复制**:单字段复制、全部复制为 JSON。
- **真实读取**:经 exiftool(family-0 分组)读取;读取失败返回空、不影响浏览。

## Exif 擦除 / 隐私清理(M4)

- **入口**:Exif 抽屉底部「擦除元信息」,居中模态弹窗。
- **应用范围**:当前图片 / 当前组 / 整个文件夹(组、夹走批处理,见下)。
- **预设**:隐私 / 分享 / 完全清理 / 保留版权 / 自定义。
- **分类勾选**:GPS 位置、设备/序列号、图像 ID、拍摄时间、软件信息、缩略图、描述/说明、作者/版权;可叠加 **自定义标签**(自由输入,无效项忽略并提示)。
- **预览与验证**:擦除前显示差异(将移除 / 保留哪些字段);擦除后重读验证字段确已移除。
- **输出(安全默认)**:默认 **导出新文件**(`-noexif` 后缀,同目录,冲突自动递增,原图即备份);或 **就地覆盖**(红色警告 + 红色确认按钮二次确认;批量就地覆盖再加一次「再次点击确认」)。
- 连缩略图 IFD(IFD1)中的描述/作者/版权副本一并清除。

## 批处理任务(M4)

- **批处理页**:启动批量擦除后自动进入(工具栏也有任务入口,运行中带指示点)。显示进度条、已完成/失败数、当前文件、错误列表。
- **任务控制**:暂停 / 继续 / 取消 / 重试失败项;**复制日志** 或 **导出日志** 到文件;可清除已完成任务。
- 当前组 / 整文件夹批量擦除接入任务调度器(串行执行,文件之间可暂停/取消)。

## Exif 自动模式(M4,会话级)

- **首张擦除后提示**:对某文件夹第一张图片擦除成功后,弹出「应用到其他图片?」——仅当前 / 当前文件夹全部(走批处理)/ 本文件夹本会话自动应用。
- **会话自动应用**:开启后在该目录导航到未处理图片时,**自动导出去元信息副本**(一律 export-new、永不覆盖原图,故浏览触发安全)。
- **防误操作**:顶部琥珀色状态条显示当前规则与目录,**一键关闭**;提示每个文件夹一个会话只弹一次。
- (永久规则、跨重启「存为默认规则」顺延 M7。)

## 文件操作

- 移到回收站(`Delete`/`Backspace`)、复制图片(`Cmd/Ctrl+C`)、复制路径(`Cmd/Ctrl+Shift+C`)、在文件管理器中显示;均作用于焦点图。
- 删除后队列自动收敛,当前组自动补齐后续图片。

## 全屏与窗口

- 全屏切换(`F11`);macOS 隐藏式标题栏(交通灯)。

## 键盘快捷键

| 快捷键 | 功能 |
|---|---|
| `←` / `→`(或 `A` / `D`) | 上一张 / 下一张(组内步进,首尾跨组) |
| `↑` / `↓` | 上一组 / 下一组 |
| `Shift+←` / `Shift+→` | 上一组 / 下一组 |
| `Home` / `End` | 第一张 / 最后一张 |
| `1`–`4` | 选择对应格子为焦点(多图) |
| `Tab` / `Shift+Tab` | 焦点切到下/上一格(组内循环,多图) |
| `V` | 在单/双/三/四宫格间循环 |
| `Ctrl/Cmd+1..4` | 直接切到单/双/三/四宫格 |
| `Enter` / `Esc` | 焦点图放大为单图 / 返回网格(多图) |
| `S` | 同步缩放开关(多图) |
| `I` | Exif 信息抽屉开关 |
| `空格` | Fit / 原始尺寸切换 |
| `+` / `-` / `0` | 放大 / 缩小 / 原始尺寸 |
| `F` | 适应窗口 |
| `R` / `Shift+R` | 旋转 / 随机一张 |
| `F11` | 全屏切换 |
| `Delete` / `Backspace` | 移到回收站 |
| `Cmd/Ctrl+C` / `Cmd/Ctrl+Shift+C` | 复制图片 / 复制路径 |
