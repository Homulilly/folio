# Folio 已实现功能

> 本文记录 **当前代码已落地** 的功能(M1 基础看图 · M2 多图并列 · M3 Exif 查看 · M4 Exif 擦除/批处理/自动模式 · M5 保存到目标/快速保存/批量重命名 · M6 格式转换 · M7 缓存/缩略图/预览显示/持久化设置/队列虚拟化/大图策略)。
> 完整需求/路线图见 [`prd.md`](./prd.md),里程碑进度见 [`mvp-tasks.md`](./mvp-tasks.md)。

## 打开与队列

- **打开方式**:打开文件、打开文件夹、拖拽文件/文件夹到窗口。
  - 打开单张图片 → 扫描其所在文件夹并定位到该图。
  - 打开文件夹 → 非递归扫描该目录下受支持的图片。
  - 拖拽多个路径 → 只收其中受支持的图片。
- **队列侧栏**:左侧图片列表,显示 **缩略图**(sharp 生成的 webp 变体,经缓存;懒加载仅渲染可视行,解码失败回退为格式徽标)、文件名、大小;点击切换;当前项高亮,多图模式下当前组整组高亮。
  - 侧栏可 **收起/展开**(工具栏最左侧的侧栏按钮),收起后画布占满宽度;**默认收起**,启动时的状态可在设置页「浏览」里改。
  - **刷新文件列表**:侧栏标题栏的刷新按钮重新扫描当前文件夹(新增/删除的图片即时反映),并 **保持停留在当前图片**。
- **文件夹浏览**:侧栏标题栏的「浏览文件夹」按钮展开目录浏览面板(默认定位到当前文件夹的 **上一级**,因此 **同级文件夹** 一目了然,当前文件夹高亮)。
  - 每个文件夹显示其 **直接图片数**;`..` 行回到上一层,`›` 钻入子文件夹。
  - 点击有图片的文件夹 **直接载入** 其图片;无图片但有子文件夹的则钻入浏览。
- **载入下一文件夹**:在最后一张图片继续按「下一张」(`→`/`D`)且未开启循环时,若存在 **下一个含图片的同级文件夹**,弹出确认提示,确认即载入该文件夹。
- **最近文件夹**:记录并可清空(主进程持久化)。
- **队列标题栏**:精简为图标控件 —— **排序**(下拉菜单)、**浏览文件夹**、**刷新**。
- **排序**:标题栏排序图标弹出下拉菜单,当前排序带勾选;支持名称、修改时间、创建时间、大小、格式(升/降序);切换排序后保持当前焦点图。

## 单图查看

- **适应窗口(Fit)**:按 **宽高同时计算** 缩放,确保整图完整可见,且不超过原始尺寸(100%);90°/270° 旋转时按旋转后的边界计算。
- **缩放**:
  - 工具栏按钮、`+/-`、`0`(原始尺寸)、`空格`(Fit/原始切换)。
  - **鼠标滚轮缩放**:滚轮即缩放,按当前显示比例平滑增减;从 Fit 进入缩放时 **从实际适应比例开始**(超大图不会突跳到 100%);对每帧滚轮增量 **削峰**,高速 / 自由滚动的滚轮也是匀速斜坡而非剧烈跳变。
- **平移**:放大后按住鼠标拖动平移,横向纵向均可。
- **双击回到 Fit**:图片放大(非 Fit)时,双击图片回到适应窗口。
- **拖出文件**:图片处于 Fit 时,从图片向外拖动即「拿起这张文件」——可拖到访达/资源管理器复制,或拖到网页上传控件;拖出的是真实文件路径(经系统原生拖放,非 `gv-img://`/base64)。放大状态下拖动仍是平移,避免误触。详见 [`file-drag-out.md`](./file-drag-out.md)。
- **旋转**:右转(整圆箭头图标);旁边的 **重置方向** 按钮在旋转后才激活。
- **切换图片箭头**:鼠标移动到画布 **左右边缘附近** 时浮现半透明箭头,点击上一张/下一张;到首/尾时对应箭头隐藏(开启循环则不隐藏)。
- **自定义滚动条**:浅灰色、半透明、仅滑块,悬浮在图片之上;隐藏原生滚动条,滚动条出现/消失不引起布局变动;可拖动。队列侧栏使用同款滚动条。

## 多图并列(差异化核心)

- **模式**:单图 / 双图 / 三图 / 四宫格;工具栏按钮、`V` 循环、`Ctrl/Cmd+1..4` 切换。
- **布局变体**:双图(横/竖)、三图(左大右二 / 三列等宽)、四宫格 2×2;有变体的模式可用工具栏「Swap layout」切换。
- **分组浏览**:按模式的图片数成组步进;末尾不足时空白格显示「No more images」;可开启 **循环浏览**。
  - **网格内滚轮**:在多列网格上滚动滚轮 **成组切换**(下一组/上一组,带节流)。
- **焦点图**:点击格子、数字键 `1-4`、`Tab`/`Shift+Tab` 选择焦点,焦点格蓝色边框;焦点图驱动状态栏与文件操作。
- **焦点单图**:`Enter` 或 **双击格子** 将焦点图临时放大为单图;**再次 `Enter` 或在单图上双击** 返回网格(双击与回车互为进入/返回);`Esc` 或 **右下角半透明返回按钮** 也可返回。进入/返回都重置为 Fit。(单图已放大时,双击优先回到 Fit;再双击才返回网格。)
- **缩放联动**:`S` 或工具栏开关切换 **同步缩放**——开启时所有格同步缩放;**关闭时每格独立记住自己的缩放**,切换焦点不会把上一格的缩放带到新格(各格互不影响)。
- **旋转**:多列下右转 **对所有格生效**,并 **带入焦点单图**(共享旋转状态)。
- **状态栏**:显示当前模式、组范围(第 X–Y 张 / 共 N)、焦点图文件名/尺寸/格式/缩放、Sync 指示。
- **切换重置**:切换视图数量(单/双/三/四)、进出焦点单图时,视图重置为 Fit。

## 格式与元信息

- **真实格式识别**:按 **magic bytes(文件头)** 判断真实格式,而非扩展名——扩展名说谎的图片(如 `.png` 实为 JPEG)也能正确判定与显示。
  - 覆盖 JPEG/PNG/GIF/WebP/BMP/TIFF/AVIF/HEIC/HEIF/JPEG XL;纯检测逻辑在 `@folio/image-processing`(含单测)。
  - `gv-img://` 协议据此返回正确的 `Content-Type`;渲染层据此判断可渲染性与格式标签。
- **渲染范围**:Chromium 原生可解码的格式直接显示;**其余格式(TIFF 等)经 sharp 生成的 `preview` 变体显示**(M7),不再只是占位。**HEIC/HEIF/JPEG XL**:预置 libvips 不带 HEVC、也不解 JXL,**macOS 上经系统 `sips` 兜底转码**后显示预览(macOS 14+ 的 ImageIO 原生支持这三种);**Windows 暂不支持**(显示占位/解码失败提示)。SVG/ICO 仍走原图(sharp 不栅格化)。
- **加载失败按原因提示**:载入失败时回探文件状态,区分 **文件不存在** / **无法读取(权限)** / **无法解码**,单图与多图格子均给出对应文案。

## Exif 查看(M3)

- **Exif 抽屉**:右侧抽屉(工具栏信息按钮或 `I` 开关);多图模式下 **跟随焦点图**。
- **三视图**:
  - **概览(Summary)**:相机 / 镜头 / 曝光 / GPS 等要点,另含 **描述 / 作者 / 版权 / 注释 / 唯一 ID**(描述 / 作者 / 版权 在主值即 IFD0 为空时,回退读取缩略图 IFD1 的 0x010E / 0x013B / 0x8298)。**GPS 不显示坐标**,改为橙色「包含 GPS 信息」徽标,坐标在 hover 时显示。
  - **分组(Grouped)**:按 EXIF·GPS·XMP·IPTC·File 等分组;**字段名按界面语言显示中文**(原始标签名见 hover / 原始视图)。
  - **原始(Raw)**:完整的 `{ 分组: { 字段: 值 } }` **JSON**,可选中复制。
- **搜索**:按字段名或值过滤;输入分组名展开整组(分组 / 原始视图均生效)。
- **复制**:单字段复制、全部复制为 JSON。
- **真实读取**:经 exiftool(family-0 分组)读取;读取失败返回空、不影响浏览。**仅在 Exif 抽屉 / 擦除对话框打开时才读**(扫描、队列、缩略图都不读 Exif)。

## Exif 擦除 / 隐私清理(M4)

- **入口**:Exif 抽屉底部「擦除 EXIF」(与「复制 JSON」并排),居中模态弹窗。
- **应用范围**:当前图片 / 当前组 / 整个文件夹(组、夹走批处理,见下)。
- **预设**:隐私 / 分享 / 完全清理 / 保留版权 / 自定义。
- **分类勾选**:GPS 位置、设备/序列号、图像 ID、拍摄时间、软件信息、缩略图、描述/说明、作者/版权;可叠加 **自定义标签**(自由输入,无效项忽略并提示)。
- **预览与验证**:擦除前显示差异(将移除 / 保留哪些字段);擦除后重读验证字段确已移除。
- **输出(安全默认)**:默认 **导出新文件**(`-noexif` 后缀,同目录,冲突自动递增,原图即备份);或 **就地覆盖**(红色警告 + 红色确认按钮二次确认;批量就地覆盖再加一次「再次点击确认」)。
- 连缩略图 IFD(IFD1)中的描述/作者/版权副本一并清除。

## 批处理任务(M4)

- **批处理页**:启动批量擦除后自动进入(工具栏也有任务入口,运行中带指示点)。显示进度条、已完成/失败数、当前文件、错误列表。
- **任务控制**:暂停 / 继续 / 取消 / 重试失败项;**复制日志** 或 **导出日志** 到文件;可清除已完成任务。
- 当前组 / 整文件夹批量擦除接入任务调度器(串行执行,文件之间可暂停/取消)。

## Exif 自动模式(M4,会话级)

- **首张擦除后提示**:对某文件夹第一张图片擦除成功后,弹出「应用到其他图片?」——仅当前 / 当前文件夹全部(走批处理)/ 本文件夹本会话自动应用。
- **会话自动应用**:开启后在该目录导航到未处理图片时,**自动导出去元信息副本**(一律 export-new、永不覆盖原图,故浏览触发安全)。
- **防误操作**:顶部琥珀色状态条显示当前规则与目录,**一键关闭**;提示每个文件夹一个会话只弹一次。
- **记住默认规则(M7)**:擦除对话框会记住上次用的预设 / 勾选分类 / 自定义标签并持久化,下次开框自动预填(跨重启)。(全局自动应用 scope 刻意不做,避免在每个浏览过的文件夹散落导出副本。)

## 保存到目标文件夹(M5)

- **入口**:右键菜单「保存到文件夹…」始终打开完整对话框;工具栏快速保存按钮 / `T` 用记住的规则一键保存(见「快速保存」)。
- **应用范围**:当前图片 / 当前组 / 整个文件夹(组、夹走批处理任务页,保存前有任务预览)。
- **目标文件夹**:弹系统目录选择。
- **命名方式**:保持原名 / MD5+扩展名 / SHA1+扩展名 / 序号(`{nr:001}`) / 自定义模板。
  - 模板变量:`{name} {ext} {md5} {sha1} {date} {time} {mtime} {width} {height} {index} {nr:001}`;非法字符自动清理。
- **冲突处理**:追加序号 / 跳过 / 覆盖 / MD5 相同则跳过(交互式逐文件询问顺延)。
- **预览**:列出每个文件的「原名 → 目标名」(MD5/SHA1 以占位显示,执行时计算)。
- **安全**:**只复制不移动,绝不修改原图**,且拒绝写到源文件自身。

## 快速保存(M5 · 多目标 + 持久化 M7)

- 工具栏快速保存按钮 + `T` 键:一键把当前焦点图保存到 **记住的文件夹**(沿用规则的命名 / 冲突策略)。
- **首次设置**:尚无规则时按 `T` → 自动打开保存对话框;在对话框成功保存一次即建立规则。此后规则在 **设置页** 管理(见「设置与持久化」),对话框不再覆盖它。
- **多目标文件夹**:规则可包含 **多个目标文件夹**。按 `T` 时:
  - 无目标 → 打开设置对话框;
  - **一个目标** → 直接发送;
  - **多个目标** → 弹出 **文件夹选择菜单**(↑/↓ 选、`Enter` 确认、`Esc` 取消、也可点击;菜单打开时吞掉看图快捷键,←/→、A/D 等不会在底层切图)。
- **持久化(M7)**:目标文件夹、命名与冲突规则保存到 `settings.json`,**跨重启保留**。
- 快速保存按钮 tooltip 按 0 / 1 / 多个目标分别提示。

## 批量重命名(M5)

- **入口**:右键菜单「批量重命名…」;作用于 **当前文件夹** 的图片(按当前显示排序)。
- **三种模式**:
  - **替换 / 删除**:查找替换(支持正则、大小写敏感、可仅作用文件名不含扩展名);删除即替换为空。
  - **按位置删除**:删前 N 个 / 删后 N 个 / 从第 N 位删 M 个 / 标记之前删 / 标记之后删。
  - **序号**:`前缀+分隔符+序号`(如 `Nr:001`),可设起始号、位数、是否保留扩展名。
- **实时预览表(即 dry-run)**:逐行「原名 → 新名」;冲突标红 —— 非法字符、重名(多项映射到同一名)、与已存在文件冲突;有阻塞冲突时禁用执行。
- **循环安全**:A→B、B→A 这类互换由执行期 **两阶段临时改名** 处理,不误判为冲突。
- **安全**:失败回滚已完成项、不破坏未处理文件;结果可 **复制日志 / 导出日志**。

## 格式转换(M6)

- **入口**:工具栏转换按钮 / 右键菜单「格式转换…」。
- **打开即预检**:若当前焦点图的格式 **本系统无法作为转换输入**(ICO,以及 Windows 上的 HEIC / JPEG XL),点击转换时直接提示,不再让用户选完参数才失败。
- **应用范围**:当前图片(直接) / 当前组 / 整个文件夹(组、夹走批处理任务页,带预览,可取消/重试)。
- **输出格式**:JPEG / PNG / WebP / AVIF / TIFF。
- **参数**:质量(JPEG/WebP/AVIF/TIFF,**滑块 + 数字框**)、渐进式(JPEG)、压缩等级(PNG)、无损(WebP)、速度档 effort(AVIF,与质量同一行);保留 Exif / 保留 ICC / 保留透明通道(支持 alpha 的格式)。
- **输出位置**:与原图同目录 / 输出到选定文件夹;冲突 追加序号 / 跳过 / 覆盖。
- **预览**:列出「原名 → 目标名(换扩展名)」。
- **安全**:**只写新文件,绝不覆盖原图**(原地同扩展名会自动改名);单图转换成功后 **自动在文件管理器中显示** 新文件。
- 由 sharp(libvips)在主进程异步转换,不阻塞界面。

## 文件操作

- 移到回收站(`Delete`/`Backspace`)、复制图片(`Cmd/Ctrl+C`)、复制路径(`Cmd/Ctrl+Shift+C`)、在文件管理器中显示;均作用于焦点图。
- 删除后队列自动收敛,当前组自动补齐后续图片。

## 缓存与性能(M7)

- **缩略图 / 预览缓存**:`gv-img://` 的 `thumb`(256px)/ `preview`(2048px)变体由 sharp 生成 webp,落盘到用户缓存目录并在 SQLite(`better-sqlite3`)中建索引;键含路径+尺寸+mtime,**原图改动即失效重建**。生成在主进程异步进行(sharp 自带 libvips 线程,不阻塞界面),并对并发数限流、对相同请求去重。
- **缓存淘汰**:每个变体有磁盘字节上限(默认缩略图 1024MB / 预览 2048MB,**可在设置页调整**),超额按最近最少使用(LRU)淘汰。
- **Exif / 哈希持久化**:Exif 摘要与 MD5/SHA1 在内存热缓存之外再落 SQLite,**跨重启保留**——再次查看/保存已见过的图不必重跑 ExifTool / 重算哈希。
- **单图预加载**:浏览时预热前后相邻图(各保留至多 2 个,内存有界),上一张/下一张切换更快。
- **单图一律原图**:单图视图始终载入 **完整原图**(全质量),不做预览替换;仅浏览器无法解码的格式(HEIC/TIFF 等)显示 sharp 预览,此时真实尺寸经 `image.dimensions`(sharp 读文件头,不全解码)获取,使状态栏尺寸 / 缩放百分比正确。
- **多图网格省内存**:网格每格按尺寸择源——**大图栅格(>2048px)载入轻量预览**,避免一组多张大图同时全解码;小图栅格、SVG/ICO 用原图,无法解码的格式用预览。展开某格进单图时仍显示完整原图。
- **任务历史持久化**:批处理任务完成后存入 SQLite(上限 200 条),重启后批处理页仍列出;恢复的历史任务可查看 / 导出日志,但不能重试(运行期状态已不在)。

## 设置与持久化(M7)

- **设置文件**:用户设置写入主进程 `settings.json`(原子写),启动时先读取再渲染界面(语言无闪烁)。
- **语言**:界面语言(简体中文 / 英文)即时切换并持久化;首次启动按系统语言;旧版 `localStorage` 语言设置自动迁移。
- **设置页分区**:
  - **语言**;
  - **浏览**:启动时收起侧栏、循环浏览、同步缩放开关(后两者也可从工具栏/快捷键切换,均会记住);
  - **安全**:删除到回收站前是否确认;
  - **缓存**:缩略图 / 预览缓存大小(MB);
  - **快速保存**:命名方式(保持原名 / MD5 / SHA1 / 序号 / 自定义模板)、冲突策略、**目标文件夹列表**(增删多个);
  - **诊断**:打开日志文件夹(应用日志 + 崩溃转储)。
- **自动记住的偏好**:排序方式、多图默认模式、循环/同步缩放、快速保存规则、默认擦除规则——从各自的控件操作即记住,跨重启保留。
- **诊断与崩溃日志**:未捕获异常、渲染/子进程崩溃记入本机 `userData/logs/main.log`,原生崩溃由系统崩溃报告器留存本地——**全部离线,绝不上传**;设置页一键打开日志文件夹。

## 打包(M7)

## 全屏与窗口

- 全屏切换(`F11`);macOS 隐藏式标题栏(交通灯)。
- **沉浸式全屏**:看图时进入全屏,图片区域占满屏幕,标题栏/工具栏(上)、状态栏(下)、队列栏(左)隐藏,鼠标靠近对应边缘才滑出显示;离开即收起。全屏状态(含 macOS 绿钮 / `Ctrl+Cmd+F` 触发)由主进程同步,设置页/批处理页/空状态下不进入沉浸模式以免控件被藏住。
