# Folio 已实现功能

> 本文简要记录当前代码已落地的功能。完整需求见 [`prd.md`](./prd.md)，里程碑进度见 [`mvp-tasks.md`](./mvp-tasks.md)。

## 打开与浏览

- 支持打开图片、打开文件夹，以及拖拽图片/文件夹到窗口。
- 打开单张图片时会载入其所在文件夹，并定位到该图片。
- 文件夹扫描为非递归扫描；多路径拖入时只载入受支持的图片。
- 左侧队列显示缩略图、文件名和大小，支持当前项高亮、点击切换、收起/展开。
- 队列支持刷新、排序、文件夹浏览和最近文件夹记录。
- 到达文件夹末尾后，可提示载入下一个包含图片的同级文件夹。

## 单图查看

- 支持适应窗口、原始尺寸、放大缩小、鼠标滚轮缩放和拖拽平移。
- 支持双击回到适应窗口、旋转、重置方向、上一张/下一张浮动箭头。
- 支持从图片区域把真实文件拖出到系统文件管理器或上传控件。
- 自定义滚动条覆盖在图片上，不影响画布布局。

## 多图并列

- 支持单图、双图、三图和四宫格模式。
- 支持不同布局变体，例如双图横/竖、三图不同排列、四宫格 2x2。
- 支持按组浏览、循环浏览、网格滚轮切组。
- 支持焦点格选择、焦点图临时放大为单图，以及返回网格。
- 支持同步缩放；关闭同步缩放时，每个格子独立记住缩放状态。
- 多图状态栏显示当前模式、组范围、焦点图信息和同步缩放状态。

## 格式与显示

- 使用文件头识别真实图片格式，而不是只相信扩展名。
- 支持 JPEG、PNG、GIF、WebP、BMP、TIFF、AVIF、SVG、ICO、HEIC/HEIF 和 JPEG XL 的识别或展示策略。
- Chromium 可直接解码的格式显示原图；TIFF 等格式显示生成的预览图。
- macOS 可通过系统能力预览 HEIC、HEIF 和 JPEG XL；Windows/Linux 暂不支持这些格式的预览。
- 载入失败时会区分文件不存在、权限问题和无法解码。

## Exif 查看

- 右侧 Exif 抽屉可查看当前焦点图的元信息。
- 支持概览、分组和原始 JSON 三种视图。
- 概览显示相机、镜头、曝光、描述、作者、版权、GPS 等关键字段。
- GPS 在概览中只提示“包含 GPS 信息”，避免直接暴露坐标。
- 支持字段搜索、单字段复制和全部复制为 JSON。
- Exif 只在打开抽屉或擦除对话框时读取，不参与扫描和缩略图生成。

## Exif 擦除与隐私清理

- 支持擦除当前图片、当前组或整个文件夹。
- 提供隐私、分享、完全清理、保留版权和自定义预设。
- 可按类别选择要移除的元信息，也可输入自定义标签。
- 擦除前显示差异预览，擦除后重新读取并验证结果。
- 默认导出新文件，不覆盖原图；就地覆盖需要额外确认。
- 支持会话级自动模式，在同一文件夹内自动导出清理后的副本。

## 批处理任务

- 批量擦除、批量保存和批量转换进入任务页执行。
- 任务页显示进度、当前文件、完成数、失败数和错误日志。
- 支持暂停、继续、取消、重试失败项、复制日志和导出日志。
- 已完成任务会持久化为历史记录，重启后仍可查看。

## 保存与快速保存

- 支持把当前图片、当前组或整个文件夹复制到目标文件夹。
- 支持保持原名、MD5、SHA1、序号和自定义模板命名。
- 支持追加序号、跳过、覆盖、MD5 相同则跳过等冲突策略。
- 保存操作只复制文件，不移动或修改原图。
- 快速保存可用工具栏按钮或 `T` 键触发，并支持多个记住的目标文件夹。

## 批量重命名

- 支持对当前文件夹图片进行批量重命名。
- 提供替换/删除、按位置删除和序号三类规则。
- 执行前提供实时预览，并标出非法字符、重复名称和目标冲突。
- 执行时使用两阶段临时改名，支持 A/B 文件名互换。
- 失败时回滚已完成项，避免破坏未处理文件。

## 格式转换

- 支持转换为 JPEG、PNG、WebP、AVIF 和 TIFF。
- 支持当前图片、当前组和整个文件夹范围。
- 支持质量、压缩等级、渐进式 JPEG、WebP 无损、AVIF effort、保留 Exif/ICC/透明通道等选项。
- 支持输出到原目录或指定文件夹，并提供冲突策略和结果预览。
- 转换只写新文件，不覆盖原图；单图转换成功后可在文件管理器中显示新文件。

## 文件操作

- 支持移到回收站、复制图片、复制路径、在文件管理器中显示。
- 文件操作作用于当前焦点图。
- 删除后队列会自动收敛，并尽量补齐当前组。

## 缓存与性能

- 缩略图和预览图会生成 WebP 变体并写入本地缓存。
- 缓存通过 SQLite 建索引，原图修改后会自动失效重建。
- 缓存大小可在设置页调整，并按 LRU 策略淘汰。
- Exif 摘要和 MD5/SHA1 哈希会持久化缓存，减少重复计算。
- 单图视图优先加载完整原图；多图网格会对大图使用轻量预览以降低内存占用。
- 队列侧栏使用虚拟化，只渲染可见行。

## 设置与诊断

- 设置写入本地 `settings.json`，启动时先读取再渲染界面。
- 支持简体中文和英文界面，并可即时切换。
- 设置页包含浏览、安全、缓存、快速保存和诊断选项。
- 自动记住排序方式、多图默认模式、循环/同步缩放、快速保存规则和默认擦除规则。
- 崩溃日志和应用日志只保存在本机，不上传。

## 打包与平台

- 已支持未签名 macOS DMG 构建，并验证可启动。
- macOS Developer ID 签名、公证、Windows/Linux 产物和应用图标仍待补齐。

## 全屏与窗口

- 支持 `F11` 全屏切换和 macOS 隐藏式标题栏。
- 看图时进入全屏会启用沉浸式布局，工具栏、状态栏和队列栏会按鼠标位置滑出显示。

## 键盘快捷键

| 快捷键 | 功能 |
|---|---|
| `←` / `→` 或 `A` / `D` | 上一张 / 下一张 |
| `↑` / `↓` | 上一组 / 下一组 |
| `Shift+←` / `Shift+→` | 上一组 / 下一组 |
| `Home` / `End` | 第一张 / 最后一张 |
| `1`-`4` | 选择多图焦点格 |
| `Tab` / `Shift+Tab` | 切换多图焦点 |
| `V` | 切换单图/双图/三图/四宫格 |
| `Ctrl/Cmd+1..4` | 直接切换视图数量 |
| `Enter` | 焦点单图与网格切换 |
| `Esc` | 返回网格或关闭 Exif 抽屉 |
| `S` | 同步缩放开关 |
| `I` | Exif 抽屉开关 |
| `Space` | 适应窗口 / 原始尺寸切换 |
| `+` / `-` / `0` | 放大 / 缩小 / 原始尺寸 |
| `F` | 适应窗口 |
| `R` / `Shift+R` | 旋转 / 随机一张 |
| `F11` | 全屏切换 |
| `T` | 快速保存当前图 |
| `Delete` / `Backspace` | 移到回收站 |
| `Cmd/Ctrl+C` | 复制图片 |
| `Cmd/Ctrl+Shift+C` | 复制路径 |
